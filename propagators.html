class src_dust.satpy_tools.src.propagators.SGP4SAT(elements, jd=2459714.2916666665, deg=True)#

Bases: object

Represents a satellite using the SGP4 propagation model.

This class initializes a satellite with either classical orbital elements or Two-Line Element (TLE) sets and provides methods to propagate its state.

  1. High Fidelity Propagators

src_dust.satpy_tools.src.propagators.lightyear_prop(state_0, TOF_s: float, Am_ratio=0, CD=0, CR=0, qm_ratio=0, r_obj_m=0, jd0: float = np.float64(2460709.2946328013), grav_order=7, perturbs_included=['dust'], eps=0.0001, perturbations=None) → tuple[ndarray, float, bool]#

Propagates a state vector using the 2-body problem model with various perturbations.

Parameters:

  • state_0 (np.ndarray) – The initial state vector [x, y, z, vx, vy, vz].

  • Am_ratio (float) – The area-to-mass ratio of the satellite.

  • CD (float) – The drag coefficient of the satellite.

  • CR (float) – The reflectivity coefficient of the satellite.

  • TOF_s (float) – The time of flight in seconds.

  • qm_ratio (float, optional) – The charge-to-mass ratio of the satellite. Defaults to 0.

  • r_obj_m (float, optional) – The radius of the object in meters. Defaults to 20e-6.

  • jd0 (float, optional) – The Julian date. Defaults to NOW_JD.

  • grav_order (int, optional) – The order of the spherical harmonics expansion. Defaults to 7.

  • perturbs_included (list, optional) – The list of perturbations to include. Defaults to [“dust”].

  • eps (float, optional) – The tolerance for the integration. Defaults to PROPAGATOR_EPS.

  • perturbations (Perturbations, optional) – Precomputed perturbations object. Defaults to None.

Returns:

np.ndarray: The final state vector [x, y, z, vx, vy, vz]. float: The final time. bool: A flag indicating if the satellite entered the Earth’s atmosphere.

Return type:

tuple

  1. High Fidelity Propagators, step

src_dust.satpy_tools.src.propagators.lightyear_prop_step(state_0, t_vals, Am_ratio=0, CD=0, CR=0, qm_ratio=0, r_obj_m=2e-05, jd0=2459714.2916666665, grav_order=7, perturbs_included: list = ['dust'], eps=0.0001, perturbations_class=None)#

Propagates a state vector in time with optional perturbations.

Parameters:

  • state_0 (np.ndarray) – The initial state vector [x, y, z, vx, vy, vz].

  • Am_ratio (float) – The area-to-mass ratio of the satellite.

  • CD (float) – The drag coefficient of the satellite.

  • CR (float) – The reflectivity coefficient of the satellite.

  • t_vals (np.ndarray) – The array of time values for integration.

  • qm_ratio (float, optional) – The charge-to-mass ratio of the satellite. Defaults to 0.

  • r_obj_m (float, optional) – The radius of the object in meters. Defaults to 20e-6.

  • jd0 (float, optional) – The initial Julian Date. Defaults to JD_STANDARD.

  • grav_order (int, optional) – The order of the gravitational perturbations. Defaults to 7.

  • perturbs_included (list, optional) – The list of perturbations to include. Defaults to [“dust”].

  • eps (float, optional) – The tolerance for the integration. Defaults to PROPAGATOR_EPS.

  • perturbations_class (Perturbations, optional) – Precomputed perturbations object. Defaults to None.

Returns:

np.ndarray: Array of propagated states. np.ndarray: Array of corresponding time values. int: Flag indicating if the orbit hit the ground (1) or not (0).

Return type:

tuple

Raises:

  • ValueError – If the initial state vector or time values are invalid.

  • _TerminationError – If the orbit hits the ground, leaves Earth, encounters NaNs, or becomes hyperbolic.
