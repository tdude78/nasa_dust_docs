class src_dust.satpy_tools.src.orbital_objects.KeplerObject(name, state_kep, t0)#

Bases: object

Represents an orbital object defined by Keplerian elements.

This class converts a Keplerian state vector into ECI coordinates and creates an associated KeplerianOrbit for propagation.

Parameters:

  • name (str) – The name of the object.

  • state_kep (array_like) – The state vector in Keplerian elements.

  • t0 (float) – The initial time (epoch) for the state vector.

propagate(t, update_current_info=False)#

Propagate the object’s orbit to a new time.

This method computes the true anomaly at the specified time, then determines the corresponding position and velocity in ECI coordinates. It then converts these values back to Keplerian elements. Optionally, the object’s internal state is updated.

Parameters:

  • t (float) – The target time (epoch) for propagation.

  • update_current_info (bool, optional) – If True, update the object’s internal state with the new propagated state. Defaults to False.

Returns:

The propagated state vector in Keplerian elements.

Return type:

numpy.ndarray

  1. Stationary Object

class src_dust.satpy_tools.src.orbital_objects.StationaryObject(name, state, jd=2459714.2916666665, covar_state=None, eps=0.0001)#

Bases: object

Represents a stationary orbital object with a fixed state and covariance.

The state vector is expected to have 7 elements (3 positions, 3 velocities, and 1 size), and the covariance matrix must be 7x7.

Parameters:

  • name (str) – The name of the object.

  • state (array_like) – The state vector [x, y, z, vx, vy, vz, size].

  • jd (float, optional) – The current Julian date. Defaults to JD_STANDARD.

  • covar_state (array_like, optional) – The covariance matrix of the state vector. Defaults to a zero 7x7 matrix.

  • eps (float, optional) – The epsilon value for propagation. Defaults to PROPAGATOR_EPS.

Extends StationaryObject by including additional properties such as area-to-mass ratio, drag coefficients, and perturbation settings.

arbitrary_transfer(delta_v: ndarray | float | int, t_wait_s: float, TOF_s)#

Perform an arbitrary transfer maneuver by applying a delta-v.

This method applies the specified delta-v (if scalar, in the direction of velocity), optionally propagates the state for a waiting period, and then further propagates for the remaining time of flight.

Parameters:

  • delta_v (numpy.ndarray, float, or int) – The delta-v to be applied. If a scalar, it is applied in the direction of the velocity.

  • t_wait_s (float) – The waiting time in seconds before applying the maneuver.

  • TOF_s (float or list of float) – The time of flight in seconds for subsequent propagation.

Returns:

A tuple containing: - Propagated mean state vector. - Propagated covariance matrix. - Propagated sigma points. - A new OrbitalObject instance with updated state.

Return type:

tuple

hohmann_transfer_full(final_semimajor_axis: float, t_wait_s: float = 0, TOF_post_transfer_s: float = 0)#

Perform a full Hohmann transfer maneuver.

This method executes a half transfer followed by a second impulse to circularize the orbit. An optional post-transfer propagation is performed.

Parameters:

  • final_semimajor_axis (float) – The target semimajor axis after the full transfer.

  • t_wait_s (float, optional) – The waiting time (in seconds) before executing the maneuvers. Defaults to 0.

  • TOF_post_transfer_s (float, optional) – The time of flight (in seconds) after the transfer for further propagation. Defaults to 0.

Returns:

A tuple containing: - Propagated mean state vector. - Propagated covariance matrix. - Propagated sigma points. - Total delta_v (float): Sum of both impulses. - A new OrbitalObject instance with updated state.

Return type:

tuple

hohmann_transfer_half(final_semimajor_axis: float, t_wait_s: float = 0)#

Perform a half Hohmann transfer maneuver.

This method computes the first impulse required to change the semimajor axis to the desired final value and propagates the state accordingly.

Parameters:

  • final_semimajor_axis (float) – The target semimajor axis after the transfer.

  • t_wait_s (float, optional) – The waiting time (in seconds) before executing the maneuver. Defaults to 0.

Returns:

A tuple containing: - Propagated mean state vector. - Propagated covariance matrix. - Propagated sigma points. - Delta_v1 (float): The computed impulse velocity. - A new OrbitalObject instance with updated state.

  1. Distributed Orbital Object (i.e. a dust cloud)

class src_dust.satpy_tools.src.orbital_objects.DistributedObject(name, mean, covar=None, jd=2459714.2916666665, Am_ratio=0, qm_ratio=0, CD=0, CR=0, grav_order=7, perturbs_included=['dust'], eps=0.0001, NUM_DISTS=8)#

Bases: object

Represents a distributed object (e.g., a cloud of dust) with a probabilistic state.

This class maintains a distribution over the state (mean and covariance) and provides methods for propagating the distribution.

Parameters:

  • name (str) – The name of the object.

  • mean (numpy.ndarray) – The initial mean state vector.

  • covar (numpy.ndarray, optional) – The initial covariance matrix of the state vector. Defaults to None.

  • jd (float, optional) – The initial Julian date. Defaults to JD_STANDARD.

  • Am_ratio (float, optional) – The area-to-mass ratio. Defaults to 0.

  • qm_ratio (float, optional) – The charge-to-mass ratio. Defaults to 0.

  • CD (float, optional) – The drag coefficient. Defaults to 0.

  • CR (float, optional) – The radiation pressure coefficient. Defaults to 0.

  • grav_order (int, optional) – The order of the gravity model. Defaults to 7.

  • perturbs_included (list, optional) – List of perturbations to include. Defaults to [‘dust’].

  • eps (float, optional) – The epsilon value for propagation. Defaults to PROPAGATOR_EPS.

  • NUM_DISTS (int, optional) – Number of minor distributions to use. Defaults to NUM_DISTS.

  1. Canister of Dust

class src_dust.satpy_tools.src.orbital_objects.CanisterWithDust(name, mean_dust=None, covar_dust=None, mean_canister=None, covar_canister=None, jd=2459714.2916666665, CD_canister=0, CR_canister=0, CD_dust=2.2, CR_dust=1.25, Am_ratio_canister=0, Am_ratio_dust=0, qm_ratio_canister=0, qm_ratio_dust=0, grav_order=7, perturbs_included=['dust'], eps=0.0001, num_dists=8)#

Bases: object

Represents a canister that releases dust, modeled by combining an OrbitalObject (the canister) with a DistributedObject (the dust).

Parameters:

  • name (str) – The name of the object.

  • mean_dust (numpy.ndarray, optional) – The initial mean state vector for the dust. If None, it is copied from mean_canister.

  • covar_dust (numpy.ndarray, optional) – The initial covariance matrix for the dust.

  • mean_canister (numpy.ndarray) – The initial mean state vector for the canister.

  • covar_canister (numpy.ndarray, optional) – The initial covariance matrix for the canister.

  • jd (float, optional) – The initial Julian date. Defaults to JD_STANDARD.

  • CD_canister (float, optional) – Drag coefficient for the canister. Defaults to 0.

  • CR_canister (float, optional) – Radiation pressure coefficient for the canister. Defaults to 0.

  • CD_dust (float, optional) – Drag coefficient for the dust. Defaults to 2.2.

  • CR_dust (float, optional) – Radiation pressure coefficient for the dust. Defaults to 1.25.

  • Am_ratio_canister (float, optional) – Area-to-mass ratio for the canister. Defaults to 0.

  • Am_ratio_dust (float, optional) – Area-to-mass ratio for the dust. Defaults to 0.

  • qm_ratio_canister (float, optional) – Charge-to-mass ratio for the canister. Defaults to 0.

  • qm_ratio_dust (float, optional) – Charge-to-mass ratio for the dust. Defaults to 0.

  • grav_order (int, optional) – Order of the gravity model. Defaults to 7.

  • perturbs_included (list, optional) – List of perturbations to include. Defaults to [‘dust’].

  • eps (float, optional) – Epsilon value for propagation. Defaults to PROPAGATOR_EPS.

  • num_dists (int, optional) – Number of minor distributions to use. Defaults to NUM_DISTS.

propagate_both(TOF2target_post_release: float, TOF2release: float = 0.0)#

Propagate both the canister and the dust components.

This method first propagates the canister (if a nonzero release time is specified) and then propagates the dust distribution for a given time of flight after release.

Parameters:

  • TOF2target_post_release (float) – The time of flight (in seconds) after release for propagation.

  • TOF2release (float, optional) – The time of flight (in seconds) before release. Defaults to 0.0.

Returns:

A tuple containing: - Propagated means of the dust distribution. - Propagated covariance matrices of the dust distribution. - Propagated sigma points for the dust distribution. - Weights of the dust minor distributions. - A new CanisterWithDust instance with updated state.

Return type:

tuple

class src_dust.satpy_tools.src.orbital_objects.WalkerConstellation(orbital_altitude: float | int, inc: float | int, total_sats: int, planes: int, f_relspace: float | int = None, init_raan: float | int = 0, jd0=2459714.2916666665, name='default walker constellation', covar_state0=None, r_sat=1, Am_ratio=0, qm_ratio=0, CD=0, CR=0, grav_order=7, perturbs_included=['dust'], eps=0.0001, deg=True)#

Bases: object

Represents a Walker constellation of satellites.

Parameters:

  • orbital_altitude (float or int) – The orbital altitude (in kilometers) of the satellites.

  • inc (float or int) – The inclination of the satellites.

  • total_sats (int) – The total number of satellites in the constellation.

  • planes (int) – The number of orbital planes.

  • f_relspace (float or int, optional) – The relative spacing factor between satellites. Defaults to 1 if not provided.

  • init_raan (float or int, optional) – The initial RAAN (in degrees). Defaults to 0.

  • jd0 (float, optional) – The initial Julian date. Defaults to JD_STANDARD.

  • name (str, optional) – The name of the constellation. Defaults to “default walker constellation”.

  • covar_state0 (numpy.ndarray, optional) – The initial covariance matrix for each satellite. Defaults to a 7x7 zero matrix.

  • r_sat (float, optional) – The size parameter for each satellite. Defaults to 1.

  • Am_ratio (float, optional) – The area-to-mass ratio for each satellite. Defaults to 0.

  • qm_ratio (float, optional) – The charge-to-mass ratio for each satellite. Defaults to 0.

  • CD (float, optional) – The drag coefficient for each satellite. Defaults to 0.

  • CR (float, optional) – The radiation pressure coefficient for each satellite. Defaults to 0.

  • grav_order (int, optional) – The gravitational model order. Defaults to 7.

  • perturbs_included (list, optional) – List of perturbations to include. Defaults to [“dust”].

  • eps (float, optional) – The epsilon value for propagation. Defaults to PROPAGATOR_EPS.

  • deg (bool, optional) – If True, angular parameters are interpreted in degrees. Defaults to True.
