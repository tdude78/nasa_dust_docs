src_dust.satpy_tools.src.utils.conversions.equinoc2eci(elems: ndarray, t=0.0, t0=0.0) → ndarray#

Convert equinoctial orbital elements to an ECI state vector using a C wrapper.

Parameters:

  • elems (numpy.ndarray) – Equinoctial elements of shape (6,) or (7,). If 7 elements are provided, the last element is an extra parameter (size).

  • t (float, optional) – Time offset parameter (default is 0.0).

  • t0 (float, optional) – Reference time parameter (default is 0.0).

Returns:

The resulting ECI state vector of shape (7,), with the extra element appended.

Return type:

numpy.ndarray

  1. Parse TLE

src_dust.satpy_tools.src.utils.conversions.parseTLE(line1, line2, deg, CD=2.2)#

Parse Two-Line Element (TLE) data and extract orbital parameters.

Parameters:

  • line1 (str) – The first line of the TLE.

  • line2 (str) – The second line of the TLE.

  • deg (bool) – If True, the angular elements are converted to radians.

  • CD (float, optional) – Drag coefficient (default is 2.2).

Returns:

A tuple containing:

  • elems (numpy.ndarray): An array with the elements [a, e, i, RAAN, ω, M, B_star].

  • jd (float): The Julian date corresponding to the TLE epoch.

  • norad (str): The NORAD identifier extracted from the TLE.

  • Am_ratio (float): The area-to-mass ratio derived from B_star and CD.

Return type:

tuple
